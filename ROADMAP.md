**Приоритет: КРИТИЧЕСКИЙ** | Миграции: 133-135 | Страница `/listening` — 6 уроков + 270 упражнений

Нужно создать уроки с `type = 'listening'` и упражнения трёх типов:

**1a. Диктанты (dictation)** — 120 штук (20 на уровень)
```json
prompt_json: {"word_he": "שלום", "word_translit": "шалОм", "hint": "приветствие"}
answer_json: {"correct": "שלום", "accept": ["שלום"]}
```
- L1-L2: одиночные слова, простые фразы (3-5 слов)
- L3-L4: короткие предложения (5-8 слов)
- L5-L6: полные предложения (8-12 слов), сложная лексика

**1b. Минимальные пары (minimal_pairs)** — 90 штук (15 на уровень)
```json
prompt_json: {
  "pair_a": {"he": "כלב", "translit": "кЕлев", "meaning": "собака"},
  "pair_b": {"he": "כלבה", "translit": "кальбА", "meaning": "собака (ж.)"},
  "question": "Какое слово вы слышите?",
  "correct_pair": "a"
}
```
Пары для различения:
- L1: ב/פ, כ/ח, ת/ט, ס/שׂ (глухие/звонкие)
- L2: קָ/כָּ, ע/א, ה в конце (немое), дагеш
- L3: биньяны (כָּתַב/נִכְתַּב), ударение (БОкер/бокЕр)
- L4: регистр (разговорный vs литературный), сленг
- L5-L6: арамеизмы, архаизмы, стилистические нюансы

**1c. Аудирование с вопросами (listening_comprehension)** — 60 штук (10 на уровень)
```json
prompt_json: {
  "text_he": "היום מזג האוויר יפה. השמש זורחת והשמיים כחולים.",
  "text_translit": "хайОм мезЕг хаавИр яфЕ...",
  "questions": [
    {"question": "Какая сегодня погода?", "options": ["Хорошая", "Плохая", "Дождливая"], "correct": "Хорошая"}
  ]
}
```
- L1: 2-3 предложения, 1 вопрос
- L2: 3-4 предложения, 2 вопроса
- L3-L4: абзац, 3 вопроса
- L5-L6: полный текст, 4-5 вопросов

**Итого: 6 уроков (1 на уровень) + 270 упражнений**
