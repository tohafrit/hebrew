```

L5 (27 текстов): "המאמר הנוכחי עוסק בסוגיית ЦАХАЛ и общество..."
L6 (23 текстов): "הטקסט הנוכחי מייצג את הרמה הגבוהה ביותר..."
```

- Все 80 текстов используют одно и то же предложение-каркас
- Русские слова вместо ивритских заголовков тем

### 2. Упражнения: ~334 шаблонных (11%)

**Проблема A: 78 fill_blank — одно предложение на всех**
```json
{"sentence": "ה_____ חשוב מאוד בתחום הזה (topic)", "hint": "ключевое понятие"}
```
Все 78 — идентичный каркас с подставленным словом. Студент ничего не учит.

**Проблема B: 78 multiple_choice — ответ ВСЕГДА на позиции 0**
```json
{"options": ["<правильный>", "время", "место", "человек"], "question": "Что означает <word>?"}
```
- Отвлекающие варианты всегда одинаковые ("время, место, человек")
- correct_index = 0 у 73% ВСЕХ MC-упражнений (500 из 687)
- Студент может пройти 73% просто нажимая первый вариант

**Проблема C: 78 dictation — одинаковое объяснение**
```json
{"audio_text": "המומחים דנו ב<WORD> והציגו את מסקנותיהם"}
```
Все 78 — одно предложение с подставленным словом, одинаковый explanation у всех.

**Проблема D: 146 word_order (40%) — слова не перемешаны**
`correct_order` = `[0, 1, 2, 3, ...]` — слова уже в правильном порядке. Задание бессмысленно.

**Что нужно:**
1. Удалить ~234 шаблонных упражнений (78+78+78)
2. Заменить реальными упражнениями с разнообразными формулировками
3. Рандомизировать correct_index в multiple_choice (равномерно 0-3)
4. Исправить word_order: реально перемешать слова
5. Dictation: уникальные предложения для каждого упражнения

### 3. Диалоги: 129 реплик без иврита

**Суть:** 129 из 1,366 реплик (9.4%) имеют пустой `text_he`. Все принадлежат роли "ב" (ученик). Русский перевод есть, иврит — пустая строка.

- 53 из 180 диалогов затронуты (29.4%)
- Распределение: L1 — 13.5%, L2 — 12%, L3 — 11%, L4 — 8%, L5 — 8%, L6 — 7%

**Варианты решения:**
- A) Это задумано как fill-in-the-blank → нужна поддержка на фронтенде (сейчас нет)
- B) Это баг данных → заполнить недостающий иврит

**Рекомендация:** вариант B — заполнить иврит. Если нужен режим "fill-in", сделать его отдельным типом упражнения.

### 4. Аудио: полный ноль

`audio_url` пустой у:
- Всех 200 текстов для чтения
- Всех 180 диалогов
- Всех 2,000 примеров предложений
- Всех 8,066 слов

Для языковой платформы уровня "до носителя" это критический пробел. Аудирование — один из 4 базовых навыков.

**Решение:** интеграция TTS (Google Cloud TTS или Azure) для генерации аудио по ивритскому тексту. Или отложить на отдельный этап.
